Web Optimization Project

Optimized performance and browser rendering of this very troublesome portfolio site in terms of critical rendering path and frames per second (FPS). Used several technologies and JavaScript refactoring. Using Grunt, I optmized images by resizing and compressing, and minified JS for all sites. Optimized fonts and asynchronized loading of JS. Improved Google PageSpeed Insights score from 23/100 to 95/100 for mobile and 97/100 for desktop for the main site. Optimized Cam's Pizzeria site to render at 60 FPS when scrolling and resizing pizza images in under 5ms. Used Chrome's Dev Tools Timeline to measure and record performance.
